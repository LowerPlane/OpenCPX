# OpenCPX Working Group Meetings

This directory contains meeting schedules, agendas, and notes for OpenCPX working groups.

## Meeting Schedule

### Schema Working Group
- **Frequency**: Bi-weekly, Tuesdays
- **Time**: 10:00 AM PT / 1:00 PM ET / 6:00 PM UTC
- **Duration**: 1 hour
- **Focus**: Schema evolution, extensions, backward compatibility

### SDK Working Group
- **Frequency**: Bi-weekly, Wednesdays
- **Time**: 9:00 AM PT / 12:00 PM ET / 5:00 PM UTC
- **Duration**: 1 hour
- **Focus**: SDK development, cross-language consistency

### Integrations Working Group
- **Frequency**: Monthly, First Thursday
- **Time**: 11:00 AM PT / 2:00 PM ET / 7:00 PM UTC
- **Duration**: 1 hour
- **Focus**: Platform integrations, vendor coordination

### Documentation Working Group
- **Frequency**: Monthly, Second Friday
- **Time**: 10:00 AM PT / 1:00 PM ET / 6:00 PM UTC
- **Duration**: 45 minutes
- **Focus**: Documentation improvements, tutorials

## How to Join

1. **Check the calendar** - Meeting links are posted in GitHub Discussions
2. **Add to your calendar** - Subscribe to the OpenCPX community calendar
3. **Prepare** - Review the agenda beforehand
4. **Participate** - All are welcome to attend and contribute

## Meeting Format

### Agenda (posted 24 hours before)
- Status updates from participants
- Discussion items
- Decisions needed
- Action items review

### During the Meeting
- Meetings are recorded (with consent)
- Notes are taken collaboratively
- Decisions are documented
- Action items are assigned

### After the Meeting
- Notes published to this directory
- Recording linked (if applicable)
- Action items tracked in GitHub Issues

## Proposing Topics

To add items to a meeting agenda:

1. Open a GitHub Discussion in the appropriate category
2. Tag it with `meeting-agenda`
3. Include context and what decision/input is needed
